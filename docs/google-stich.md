# Phân tích Google Stitch AI SDK (`@google/stitch-sdk`)

## Tổng quan

**Stitch SDK** là một công cụ từ Google Labs cho phép **tạo giao diện UI (mobile/web) từ text prompt** và trích xuất HTML + ảnh chụp màn hình theo chương trình.

- **Package**: `@google/stitch-sdk` (npm, v0.3.5)
- **License**: Apache 2.0
- **Nền tảng**: Node.js — tương thích hoàn toàn với backend hiện tại

---

## Các tính năng chính

| Tính năng | Mô tả |
|---|---|
| **Generate Screen** | Tạo UI screen từ text prompt → nhận HTML + screenshot URL |
| **Edit Screen** | Chỉnh sửa UI đã tạo bằng prompt mới |
| **Generate Variants** | Tạo nhiều biến thể (layout, color, font, images) |
| **Design System** | Quản lý theme/branding cho project |
| **Device Types** | `MOBILE`, `DESKTOP`, `TABLET`, `AGNOSTIC` |
| **Model Selection** | `GEMINI_3_PRO`, `GEMINI_3_FLASH` |
| **Tool Client** | Direct MCP tool access cho agent/orchestration |
| **AI SDK Integration** | Tích hợp Vercel AI SDK (`stitchTools()`) |
| **ADK Integration** | Tích hợp Google Agent Development Kit |

---

## Authentication

Cần một trong hai cách:
- **`STITCH_API_KEY`** (env var hoặc explicit)
- **`accessToken` + `projectId`** (Google Cloud)

→ **Phù hợp hoàn toàn với hệ thống Global Settings** đã thiết kế trong ROADMAP — chỉ cần thêm `stitch_api_key` vào cấu hình.

## API cốt lõi

```js
import { stitch } from "@google/stitch-sdk";

// Tạo project (thông qua Tool Client)
const result = await stitch.callTool("create_project", { title: "My App" });

// Generate screen
const project = stitch.project("project-id");
const screen = await project.generate("A login page with email and password");
const html = await screen.getHtml();       // Download URL → HTML
const imageUrl = await screen.getImage();  // Download URL → screenshot

// Edit screen
const edited = await screen.edit("Make background dark, add sidebar");

// Variants
const variants = await screen.variants("Try different colors", {
  variantCount: 3,
  creativeRange: "EXPLORE",
  aspects: ["COLOR_SCHEME", "LAYOUT"]
});
```

## Đề xuất tích hợp vào Workflow

### Node mới: `stitch_ui_gen` (Stitch UI Generator)

| Thuộc tính | Chi tiết |
|---|---|
| **Inputs** | Text prompt (trực tiếp hoặc từ node trước), Device Type, Model ID |
| **Outputs** | `htmlUrl`, `imageUrl`, `screenId`, `projectId` |
| **Config** | `STITCH_API_KEY` từ Global Settings |

### Luồng workflow ví dụ:

```
[Meta Chat / LLM Node] → prompt → [Stitch UI Gen] → htmlUrl + imageUrl
```

### Các tính năng nâng cao có thể thêm (Roadmap Mở rộng):
1. **Edit Screen node** — nhận `screenId` + edit prompt → UI chỉnh sửa
2. **Variant Generator** — tạo nhiều phiên bản UI từ 1 screen
3. **Design System node** — áp dụng brand/theme lên screens

### Backend handler mẫu:

```js
// node-handlers/stitch-ui-gen.handler.js
const { Stitch, StitchToolClient } = require('@google/stitch-sdk');

async function handle(node, inputs, context) {
  const { log } = context;
  const apiKey = context.globalSettings?.stitch_api_key;
  
  if (!apiKey) {
    throw new Error('Thiếu STITCH_API_KEY trong Global Settings');
  }

  const client = new StitchToolClient({ apiKey });
  const sdk = new Stitch(client);
  
  const prompt = node.data.prompt || inputs.find(i => i.text)?.text;
  const deviceType = node.data.deviceType || 'DESKTOP';
  
  // Tùy chọn: Sử dụng project có sẵn hoặc tạo project mới ẩn danh
  let projectId = node.data.projectId;
  if (!projectId) {
    const res = await client.callTool("create_project", { title: "Generated by Vibes AI" });
    projectId = res.id; 
  }

  const project = sdk.project(projectId);
  const screen = await project.generate(prompt, deviceType);
  
  const htmlUrl = await screen.getHtml();
  const imageUrl = await screen.getImage();
  
  await client.close();
  
  return { 
    htmlUrl, 
    imageUrl, 
    screenId: screen.id, 
    projectId: project.id,
    imagePreview: imageUrl // Cho phép canvas preview ảnh
  };
}
```
